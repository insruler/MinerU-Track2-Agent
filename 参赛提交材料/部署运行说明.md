## 一、系统架构概览

```
惯导智衡 Data Agent
├── app.py              # FastAPI 主程序（单文件启动）
├── src/
│   └── index.html      # 前端 Web UI（单页应用）
├── data/
│   ├── knowledge.db    # SQLite 知识库（自动创建）
│   └── pdfs/           # 上传的 PDF 文件（自动创建）
├── logs/
│   └── app.log         # 运行日志
└── docs/               # 文档目录
```

## 二、运行环境要求

### 操作系统
- Linux（推荐 Ubuntu 20.04+）
- macOS 12+
- Windows 10+（需 WSL2 或原生 Python）

### Python 版本
- Python 3.10+（推荐 3.12）

### 硬件资源
- CPU：2核+
- 内存：2GB+（推荐 4GB）
- 磁盘：10GB+（用于 PDF 存储和数据库）
- 网络：需访问 MinerU API（mineru.net）和 MiniMax API

### 依赖库

```
fastapi>=0.110.0
uvicorn>=0.29.0
requests>=2.31.0
python-multipart>=0.0.9
starlette>=0.36.0
```

安装命令：
```bash
pip install fastapi uvicorn requests python-multipart starlette
```

## 三、快速启动

### 方式一：直接启动（推荐）

```bash
# 1. 克隆代码
git clone https://github.com/[your-repo]/mineru-track2-agent.git
cd mineru-track2-agent

# 2. 安装依赖
pip install fastapi uvicorn requests python-multipart starlette

# 3. 配置 API Key（可选，已内置默认值）
export MINIMAX_API_KEY="your_minimax_api_key"
export MINERU_TOKEN="your_mineru_token"

# 4. 启动服务
python3 app.py
```

服务启动后访问：http://localhost:7883

### 方式二：后台运行

```bash
nohup python3 app.py > logs/app.log 2>&1 &
echo "PID: $!"
```

### 方式三：使用已部署的线上服务

**服务地址**：http://49.232.174.229:7883

无需本地部署，直接通过浏览器访问即可使用全部功能。

## 四、API 接口说明

### 基础信息

- **Base URL**：`http://49.232.174.229:7883`
- **协议**：HTTP/1.1
- **认证**：无需认证（开放访问）
- **数据格式**：JSON

### 接口列表

#### 1. 系统状态

```
GET /api/stats
```

**返回示例**：
```json
{
  "entities": 701,
  "relations": 704,
  "documents": 56
}
```

---

#### 2. 批量 PDF 解析

```
POST /api/parse/batch
Content-Type: multipart/form-data
```

**参数**：
- `files`：PDF 文件列表（multipart）

**返回示例**：
```json
{
  "job_id": "job_1716288000000",
  "total": 3,
  "message": "已提交批量解析任务"
}
```

---

#### 3. 查询解析任务状态

```
GET /api/jobs/{job_id}
```

**返回示例**：
```json
{
  "job_id": "job_1716288000000",
  "status": "running",
  "total": 3,
  "completed": 2,
  "failed": 0,
  "tasks": [
    {
      "job_id": "job_1716288000000_0",
      "filename": "report.pdf",
      "status": "done",
      "progress": 100,
      "chars": 26743,
      "entities_count": 45,
      "relations_count": 38
    }
  ]
}
```

**任务状态说明**：
| 状态 | 含义 |
|------|------|
| pending | 等待处理 |
| parsing | MinerU 解析中 |
| extracting | LLM 实体抽取中 |
| done | 完成 |
| failed | 失败（error 字段含错误信息） |

---

## 七、常见问题

**Q：服务启动后无法访问？**
A：检查防火墙/安全组是否放行 7883 端口。腾讯云需在控制台「安全组」添加入站规则：TCP 7883。

**Q：PDF 解析失败？**
A：检查 MINERU_TOKEN 是否有效（有效期约90天）。可在 mineru.net 重新申请。

**Q：LLM 调用超时？**
A：MiniMax API 有并发限制，大批量任务时可能出现超时，系统会自动记录失败状态，支持重新上传解析。

**Q：知识图谱为空？**
A：需先上传并成功解析 PDF 文档，解析完成后图谱自动更新。
